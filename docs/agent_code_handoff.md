# 4Safety — Agent Code Handoff

## Contexto

Documento operacional para execução sequencial do novo site da 4Safety.

**Objetivo:** melhorar conversão, clareza visual, SEO e estrutura comercial B2B.

## Regras de execução

* Execute em ordem.
* Não pule blockers.
* Faça commits pequenos e verificáveis.
* Valide visualmente cada etapa antes de avançar.
* Não faça mega refactor sem checkpoint.

---

## Constraints — NÃO FAZER

* Não reintroduzir CTA repetitivo de WhatsApp em múltiplas seções.
* Não usar textos de baixo contraste, principalmente cinza sobre preto.
* Não exibir CNPJ ou razão social antiga no rodapé.
* Não usar claims comerciais com SLA não garantido, como “orçamento em 1 dia”.
* Não exibir selos CA/INMETRO genericamente em todos os produtos.
* Não alterar copy institucional sem preservar a essência aprovada pelo cliente.
