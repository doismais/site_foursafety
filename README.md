## ⟠ Objetivo

Este repositório contém o site institucional e catálogo de produtos da 4Safety,
desenvolvido com o framework Astro para máxima performance e flexibilidade.

────────────────────────────────────────

## ⧉ Estrutura

O projeto segue a organização padrão do Astro:

```text
public/              # Arquivos estáticos (imagens, scripts legados)
src/
├── components/      # Componentes reutilizáveis
├── layouts/         # Layout principal (Layout.astro)
├── pages/           # Rotas do site (Astro)
└── styles/          # Arquivos CSS
astro.config.mjs     # Configuração do Astro
```

────────────────────────────────────────

## 🧞 Comandos

Todos os comandos devem ser executados a partir da raiz do projeto:

| Comando | Ação |
| :--- | :--- |
| `pnpm install` | Instala as dependências |
| `pnpm dev` | Inicia o servidor de desenvolvimento local |
| `pnpm build` | Gera o site estático para produção em `./dist/` |
| `pnpm preview` | Visualiza o build localmente |
